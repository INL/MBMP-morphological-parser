MBMP-morphological-parser
=========================

MBMP is een geheugen-gebaseerde morfologische parser voor de programmeertaal Python (http://www.python.org). 
De parser biedt de mogelijkheid om woorden te voorzien van een morfologische analyse. Lees voor details de handleiding (https://github.com/INL/MBMP-morphological-parser/blob/master/mbmp_handleiding.pdf?raw=true).
