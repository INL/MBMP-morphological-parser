class mbmp.parse.parse.RightHandRule[source]¶

Bases: nltk.parse.chart.AbstractChartRule

A rule that joins two adjacent edges together into a single combined edge. In particular, this rule states that any pair of edges that are not lexical (see is_lexical()) can be joined resulting in a new edge in which the left-hand-side is taken from the right edge and the right- hand-side is the adjunction of the left-hand-side of the left and right edge.

util Module¶

mbmp.parse.util.make_grammar(parse, mrepr='tokens-and-lemmas')[source]¶

Return a list of Productions on the basis of an output parse of L{MBMA}. MBMA returns parses in the following format:

[('V|*V', 'ver'), ('V', 'eis'), ('INFLtWB', 't')]

This is transformed into the following list of productions:

[PRE:ver -> 'ver', V -> PRE:ver V, V -> 'eis', INFL:t -> 't', V -> V INFL:t]
Args:
  • parse (list): a parse return by mbmp.MBMA.classify()
Returns:
list – a list of Productions.
