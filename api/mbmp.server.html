server Package¶

class mbmp.server.TimblServer(host='localhost', port=8080, features={}, classifier=None)¶

Bases: object

An interface to the TimblServer. The server can be questioned via the TimblClient. The host specifies the server address (localhost by default) and port specifies the server tcp communicating port. Features is a dictionary of flags and options send to the server. This can be specified in the config.py file.

kill()¶

Kill the TimblServer. Return True if the server is killed, false otherwise.

owns_server()¶

Return True if the timblserver is owned by the current user.

pid()¶

Return the pid of the timblserver associated with this instance of the class TimblServer. If no file is found, return None.

run()¶

Tries to start the TimblServer at the given host and port. If the server is already running, does nothing

started()¶

Return True if the server is started, False otherwise.
